# Trees
задача
Заглавие на проекта - Trees

Кратко описание на задачата - Задачата реализира двоично дърво за търсене, което поддържа основните операции: вмъкване на елемент, обхождане на дървото(inorder, preorder, postorder), изтриване на елемент по зададена стойност

Какъв тип дърво е реализирано - двоично наредено дърво

Използван програмен език и технологии - Програмата е написана на C#, използва .NET конзолно приложение, разработена е във Visual Studio и прилага обектно-ориентирано програмиране и алгоритми за двоични дървета за търсене.

Как се стартира програмата - чрез натискане на Ctrl+F5 можем да стартираме проекта

Пример за работа(вход/изход) - нямаме
